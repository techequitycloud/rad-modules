## 2024-05-26 - Hardcoded RDP password in VM_Migration module
**Vulnerability:** A hardcoded plaintext password (`m1grat10nc#nt#r`) was found in the `windows_vm.tf` startup script and `LAB_GUIDE.md` for a Windows VM created in the `VM_Migration` module.
**Learning:** Hardcoding credentials in Terraform configuration and provisioning scripts can expose sensitive access in the codebase. Using `random_password` and storing it in Secret Manager is preferred, though injecting the password into a startup script can still expose it via instance metadata. We must also escape `override_special` characters like `$` and use single quotes (`'`) instead of double quotes (`"`) when interpolating passwords into PowerShell scripts to prevent unintended variable expansion.
**Prevention:** Avoid defining default passwords in Terraform configurations and use `random_password` and Secret Manager by default for dynamically provisioned VMs. Use single quotes for shell script interpolations.
