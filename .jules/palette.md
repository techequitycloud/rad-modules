## 2024-05-15 - [UIMeta Group Alignment] **Learning:** Secondary or related variables (like private subnets or IP ranges) are sometimes accidentally assigned to `group=0` instead of matching their primary section's group, breaking the logical grouping in the deployment wizard. **Action:** Always cross-reference the `group=N` value with the section the variable resides in to ensure related fields appear together on the same wizard page.

## 2024-05-16 - [Missing Standard Outputs] **Learning:** Module templates sometimes drop standard outputs (like `cluster_credentials_cmd`) when they are copied, creating inconsistent operator experiences between similar modules like `Istio_GKE` and `Bank_GKE`. **Action:** Verify standard `SKILLS.md` required outputs exist when inspecting module output files.
